HabiTora サポート

バックアップ、復元、アカウント削除、問い合わせ方法をまとめています。

お問い合わせ

不具合、データ削除、ログイン、プライバシーに関する連絡は、nanairo.core@gmail.com へお願いします。確認時は、端末種別、発生画面、表示されたメッセージを添えると調査しやすくなります。

バックアップする

  1. アプリ下部の設定を開きます。
  2. 未ログインの場合は「ログインしてバックアップ」を押します。
  3. ログイン済みの場合は「今すぐバックアップ」を押します。
  4. 「Supabaseに保存したよ」と表示されたら完了です。

復元する

  1. 設定を開きます。
  2. 「詳細データ管理」を開きます。
  3. 「クラウド復元」を押します。
  4. 確認画面で「サーバーで端末を上書き」を押します。

アカウントとクラウドデータを削除する

  1. 削除したいアカウントでログインします。
  2. 設定を開きます。
  3. 「アカウント削除を開始」を押します。
  4. 確認ダイアログで削除を承認します。

この操作でSupabase上のアカウント、バックアップ、共有データが削除されます。端末内の習慣データは残るため、完全に消したい場合は端末からアプリを削除してください。

ログインできない時

  1. アプリが最新版であることを確認します。
  2. メールリンクの場合は、最新のメールだけを開きます。
  3. Google / Appleログインの場合は、ログイン画面を開き直します。
  4. 解決しない場合は、端末種別と表示メッセージを添えてお問い合わせください。
